Installation

To install, download the xpi below (right-click, save as) from your browser. Then, in Thunderbird, use Tools->Extensions->Install to install the xpi you just downloaded; restart Thunderbird.

  • aboutconfig-0.6.xpi: Version 0.6 for Mozilla Thunderbird 0.8 - 1.5 (tested with 1.5 on Linux) adds support for TB 1.5
  • aboutconfig-0.5.xpi: Version 0.5 for Mozilla Thunderbird 0.8 - 1.0 (tested with 0.9, 1.0 on Linux) adds a toolbar icon
  • aboutconfig-0.4.xpi: Version 0.4 for Mozilla Thunderbird 0.8 - 1.0 (tested with 1.0 on Linux and WinXP Pro) fixes an issue appearing with TB installer builds only
  • aboutconfig-0.3.xpi: Version 0.3 for Mozilla Thunderbird 0.8 - 1.0 (tested with 0.8, 0.9, 1.0RC) fixes a toolkit issue on Win/Mac
  • aboutconfig-0.2.xpi: Version 0.2 for Mozilla Thunderbird 0.8 - 0.9 (tested with 0.8, 0.9, 1.0RC1) adds support for TB 1.0.
  • aboutconfig-0.1.xpi: Version 0.1 for Mozilla Thunderbird 0.8 - 0.9 (tested with 0.8, 0.9)
